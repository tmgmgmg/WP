the_time();

投稿した日付

パラメータをとれる

例：the_time( "Y年n月j日 i" ); 　ｉは曜　⇒　2021年2月8日　水曜日

例：the_time( "Y-m-d" ); 日付一桁が嫌ならこっち

⇒　2021-02-08 y-m-dはめっちゃ使う

カスタムフィールド

WP内の固定・投稿ページ内で表示するためには

新規ページ⇒詳細⇒オプション　内のカスタムフィールドにチェック　

get_post_meta();

カスタムフィールドの値をとる

https://wpdocs.osdn.jp/関数リファレンス/get_post_meta

<?php $meta_values = get_post_meta($post_id, $key, $single); ?>　　3つパラメーターが必要

使い方例：

$price = get_post_meta(get_the_id() , ’価格’ , true);
<div><?php echo $price;?></div>

get_the_id()でこの記事のidを取得

'価格'は事前にカスタムフィールドで登録したもの

トップページやサイドバーで特定のカスタム投稿タイプを表示する

https://www.postit-web.com/2017/02/20/custom_post_loop/

<?php
  $args = [
    'post_type' => 'images', // カスタム投稿名が「g」の場合
    'posts_per_page' => 30, // 表示する数
  ];
  $my_query = new WP_Query($args); ?>
 
<?php if ($my_query->have_posts()): // 投稿がある場合 ?>
 
  <ul>
 
    <?php while ($my_query->have_posts()) : $my_query->the_post(); ?>
 
    <!-- ▽ ループ開始 ▽ -->
 
    <li>
      <h3><?php the_title(); ?></h3>
      <p><?php the_time(get_option('date_format')); ?></p>
    </li>
 
    <!-- △ ループ終了 △ -->
 
    <?php endwhile; ?>
 
  </ul>
 
<?php else: // 投稿がない場合?>
 
  <p>まだ投稿がありません。</p>
 
<?php endif; wp_reset_postdata(); ?>
